jQuery 'Readable XML' Plug-in

Description: A jQuery Plugin that converts large blocks of unreadable XML to readable and collapsable XML for display on a web page.

Download

Download the following files:

  • Latest version of jQuery
  • The Plugin: jquery.readableXML.js file
  • The Styles: jquery.readableXML.css file
  • The Images: jquery.readableXML.png file in the images folder (referenced by the css)

How To Use

Place the files in your website and reference them in your HTML head element. It should look like the following in your HTML head element.

<link rel="stylesheet" type="text/css" href="css/jquery.readableXML.css" />
<script src="js/jquery-1.4.4.js"></script>
<script src="js/jquery.readableXML.js"></script>

Examples

After the document is ready execute the following examples.

Simple Example:

Code:

 

var xml0 = "<test><car attr='test'><passenger>John</passenger><passenger>Jane</passenger>test\ntest</car></test>";

 

$('#exampleDiv1').readableXML(xml0);

 

Result:

Example with Collapse Events:

Code:

var xml1 = "<test><element1>contents 1</element1><element1>contents 2</element1></test>";

 

$('#exampleDiv2').readableXML(xml1, {

       onOpen : function ($element) {

              $('#resultsDiv2').text("You opened :" + $element.text());

       },

       onClose : function ($element) {

              $('#resultsDiv2').text("You closed :" + $element.text());

       }

});

 

Result:

Larger Example:

Code:
 

$.get('engadget_rss.xml', function (data) {

       $('#exampleDiv3').readableXML(data, {

              collapsed : true

       });

}, 'text');
